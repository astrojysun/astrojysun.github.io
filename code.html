Python Modules

MegaTable

A python library for constructing multiwavelength measurement tables from resolved observations of galaxies. Currently supports calculating radial profiles and user-defined aperture statistics based on either 2D images or discrete object catalogs.

SpectralCubeTools

Python tools for manipulating and analyzing radio spectral cubes. Currently supports convolving 3D cubes and 2D images, modeling noise levels and channel-to-channel correlations, and identifying emission line signals in cubes.

COConversionFactor

Python implementation of existing CO-to-H2 conversion factor prescriptions in the literature. Currently supports predicting conversion factors from metallicity and other parameters, as well as predicting metallicity from empirical scaling relations.
